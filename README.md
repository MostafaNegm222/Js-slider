# JavaScript Image Slider  

This project is a responsive and user-friendly image slider built with HTML, CSS, and JavaScript. It includes next/previous buttons and navigation bullets for seamless sliding between images.  

## Features  

- **Next/Previous Navigation**: Move through the images using intuitive buttons.  
- **Bullet Navigation**: Click on bullets to jump to a specific image.  
- **Smooth Animations**: Provides a visually appealing sliding effect.  
- **Responsive Design**: Works on all screen sizes.  

## How It Works  

1. Images are displayed in a slider format.  
2. Users can navigate through the slider using:  
   - **Next Button**: Moves to the next image.  
   - **Previous Button**: Moves to the previous image.  
   - **Bullets**: Jumps directly to a specific image.  
3. The slider updates the active bullet and ensures infinite looping for seamless transitions.  

## Installation  

1. Clone this repository:  
   ```bash  
   git clone <repository-url>  
   ```  
2. Open the project folder:  
   ```bash  
   cd <project-folder>  
   ```  
3. Open the `index.html` file in your browser.  

## Usage  

1. **Next/Previous Buttons**: Use the buttons to navigate through the images.  
2. **Bullets**: Click on any bullet to jump directly to the corresponding image.
